🌤️ M5WeatherStation

Умная метеостанция на базе M5Stick S3

📸 M5Stick S3 показывает погоду в Минске

📟 Что это?

M5WeatherStation — это компактная метеостанция на M5Stick S3. Показывает температуру, влажность, давление, ветер и прогноз погоды в выбранном городе. Всё через интернет — никаких датчиков.

🚀 Быстрый старт

  1. Прошей M5Stick через Arduino IDE
  2. Подключись к WiFi M5WeatherStation (пароль: weather123)
  3. Открой в браузере 192.168.4.1 → настрой WiFi
  4. Выбери город на странице IP-адрес/setup
  5. Готово! Экран показывает погоду 🎉
