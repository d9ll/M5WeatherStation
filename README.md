# 🌤️ M5WeatherStation

![M5Stick](https://img.shields.io/badge/M5Stick-S3-blue)
![Arduino](https://img.shields.io/badge/Arduino-IDE-00979D)
![License](https://img.shields.io/badge/License-MIT-green)

Умная метеостанция на базе **M5Stick S3** с веб-интерфейсом для настройки. Показывает температуру, влажность и прогноз погоды с красивой анимацией на экране.

---

## ✨ Возможности

- 📡 **WiFi настройка через браузер** — никакой возни с кодом
- 🌍 **Выбор любого города** через веб-интерфейс
- 🎨 **Красивая графика** — солнце, облака, дождь
- 💾 **Сохранение настроек** — после перезагрузки всё запоминается
- 🔘 **Кнопка M5** — ручное обновление погоды
- 🔄 **Автообновление** каждую минуту
