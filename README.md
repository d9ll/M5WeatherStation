# 🌤️ M5WeatherStation

Умная метеостанция на M5Stick S3 с веб-интерфейсом для настройки WiFi и выбора города. Погода берётся из интернета через бесплатный сервис wttr.in — никаких датчиков не нужно.

---

## ✨ Возможности

- Настройка WiFi через браузер — не нужно перекомпилировать код при смене сети
- Выбор любого города через красивый веб-интерфейс
- Анимированные иконки погоды — солнце, облака, дождь
- Сохранение всех настроек — после перезагрузки всё запоминается
- Кнопка M5 для ручного обновления погоды
- Автообновление каждую минуту

---

## 📦 Что нужно

- M5Stick S3
- USB-C кабель
- WiFi сеть 2.4GHz

## 🚀 Быстрый старт

### 1. Установка библиотек

В Arduino IDE: Скетч → Подключить библиотеку → Управлять библиотеками

Установи: M5Unified (остальное подтянется само)

### 2. Настройка платы

Инструменты → Плата → M5Stack → M5Stick-C-Plus-2

Проверь параметры:
- Flash Size: 8MB
- Partition Scheme: Default 8MB with spiffs
- USB CDC On Boot: Enabled

### 3. Загрузка прошивки

Скачай файлы проекта:
- M5WeatherStation.ino
- html_pages.h

Положи их в одну папку, открой .ino файл в Arduino IDE и нажми Загрузить.

### 4. Настройка WiFi

После загрузки на экране появится:
- SSID: M5WeatherStation
- Пароль: weather123
- Адрес: 192.168.4.1

Телефоном или ноутбуком:
1. Найди WiFi сеть M5WeatherStation
2. Подключись (пароль weather123)
3. Открой браузер, перейди на 192.168.4.1
4. Введи данные своей домашней WiFi сети
5. Нажми "Подключиться"

### 5. Выбор города

После подключения к домашнему WiFi открой {ip который появится на экране}/setup

1. Введи город на английском (например Moscow, London, Tokyo)
2. Выбери единицы измерения (°C или °F)
3. Нажми "Предпросмотр" для проверки
4. Нажми "Сохранить и перезагрузить"

Готово! Экран показывает погоду в твоём городе.

## 🎮 Управление

| Действие | Результат |
|----------|-----------|
| Короткое нажатие кнопки M5 | Ручное обновление погоды |
| Долгое нажатие (5 секунд) | Сброс всех настроек |

---

## 🔧 Решение проблем

**Не загружается прошивка**
Зажми BOOT → нажми RST → отпусти BOOT → нажми Загрузить

**Не вижу WiFi сеть M5WeatherStation**
Подожди 30-40 секунд после загрузки

**Показывает --°C**
Проверь название города (Moscow, а не Москва)

**Хочу сбросить настройки**
Долгое нажатие на кнопку M5 (5 секунд)
