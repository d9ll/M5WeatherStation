# 🌤️ M5WeatherStation

![M5Stick](https://img.shields.io/badge/M5Stick-S3-blue)
![Arduino](https://img.shields.io/badge/Arduino-IDE-00979D)
![License](https://img.shields.io/badge/License-MIT-green)

Умная метеостанция на базе **M5Stick S3** с веб-интерфейсом для настройки.

---

## ✨ Возможности

- 📡 Настройка WiFi через браузер
- 🌍 Выбор города через веб-интерфейс
- 🌡️ Температура, влажность, давление, ветер
- 🎨 Иконки погоды (солнце, облако, дождь, снег, гроза)
- 🔋 Отображение заряда батареи
- 🔘 Кнопка M5 для ручного обновления
- 🔄 Автообновление каждую минуту
- 💾 Сохранение всех настроек

---

## 🚀 Быстрый старт

### 1. Установка библиотек

В Arduino IDE: `Скетч → Подключить библиотеку → Управлять библиотеками`

Установи: **M5Unified**

### 2. Настройка платы
Инструменты → Плата → M5Stack → M5StickS3
Flash Size: 8MB
USB CDC On Boot: Enabled

### 3. Загрузка прошивки

1. Скачай файлы проекта
2. Открой `M5WeatherStation.ino` в Arduino IDE
3. Нажми **Загрузить**

### 4. Настройка

1. После загрузки появится точка доступа **M5WeatherStation** (пароль: `weather123`)
2. Подключись и открой **192.168.4.1**
3. Введи данные своей WiFi сети
4. Открой `IP-адрес/setup` и выбери город

---

## 🎮 Управление

| Действие | Результат |
|----------|-----------|
| Короткое нажатие M5 | Обновить погоду |
| Долгое нажатие M5 (5 сек) | Сбросить все настройки |
