# 🌤️ M5WeatherStation

![M5Stick](https://img.shields.io/badge/M5Stick-S3-blue)
![Arduino](https://img.shields.io/badge/Arduino-IDE-00979D)
![License](https://img.shields.io/badge/License-MIT-green)

Умная метеостанция на базе **M5Stick S3** с веб-интерфейсом для настройки. Показывает температуру, влажность и прогноз погоды с красивой анимацией на экране.

## ✨ Возможности

- 📡 **WiFi настройка через браузер** — никакой возни с кодом
- 🌍 **Выбор любого города** через веб-интерфейс
- 🎨 **Красивая графика** — солнце, облака, дождь
- 💾 **Сохранение настроек** — после перезагрузки всё запоминается
- 🔘 **Кнопка M5** — ручное обновление погоды
- 🔄 **Автообновление** каждую минуту

## 🚀 Быстрый старт

### 1. Установка библиотек в Arduino IDE
Скетч → Подключить библиотеку → Управлять библиотеками...

Установи:
- `M5Unified` by M5Stack
- `WebServer` (встроенная)
- `HTTPClient` (встроенная)
- `Preferences` (встроенная)

### 2. Настройки платы
Инструменты → Плата → M5Stack → M5StickS3

Проверь параметры:
- `Flash Size`: **8MB**
- `Partition Scheme`: **Default 8MB**
- `USB CDC On Boot`: **Enabled**

### 3. Загрузка HTML файлов

> ⚠️ **Важно**: Делается ТОЛЬКО при первом запуске!

1. Открой `M5WeatherStation.ino` в Arduino IDE
2. Нажми **Загрузить** (стрелка вправо)
3. Открой **Serial Monitor** (`Инструменты → Serial Monitor`)
4. Установи скорость **115200 бод**
5. Когда увидишь `=== READY TO RECEIVE connect.html ===`:
   - Открой файл `connect.html` в любом редакторе
   - Скопируй **ВСЁ** содержимое
   - Вставь в Serial Monitor
   - Напиши `END` на новой строке и нажми Enter
6. Повтори тоже самое для `setup.html`
7. Устройство перезагрузится и создаст точку доступа

### 4. Настройка WiFi

1. На телефоне/ноутбуке найди WiFi сеть **`M5WeatherStation`**
2. Пароль: **`weather123`**
3. Подключись, открой браузер, перейди по адресу **`192.168.4.1`**
4. Введи данные своей домашней WiFi сети
5. Нажми **Подключиться**

### 5. Выбор города

1. После подключения к WiFi открой **`192.168.4.1/setup`**
2. Введи название города на английском (например `Moscow`)
3. Выбери единицы измерения
4. Нажми **Предпросмотр** для проверки
5. Нажми **Сохранить**

## 🔧 Решение проблем

| Проблема | Решение |
|----------|---------|
| Не загружается прошивка | Зажми **BOOT**, нажми **RST**, отпусти **BOOT** |
| Не вижу WiFi сеть | Подожди 30 секунд после загрузки |
| Не обновляется погода | Проверь город (например `Moscow`, а не `Moskva`) |
| Ошибка `File not found` | Перезагрузи HTML через Serial Monitor |

## 📊 Данные с wttr.in

Погода берётся с бесплатного API `wttr.in`:
- Температура
- Влажность
- Описание (Sunny, Rain, Cloudy и т.д.)

Формат запроса: `http://wttr.in/Город?format=%t+%h+%C`

## 🔧 Как это загрузить

### Пошаговая инструкция:

1. **Создай папку** `M5WeatherStation` на компьютере

2. **Создай файлы** с точным названиями:
   - `M5WeatherStation.ino` — скопируй код сверху
   - `connect.html` — скопируй HTML
   - `setup.html` — скопируй HTML
   - `README.md` — скопируй текст

3. **Открой** `M5WeatherStation.ino` в Arduino IDE

4. **Выбери плату**: `M5StickS3`

5. **Нажми Загрузить**

6. **Открой Serial Monitor** (скорость 115200)

7. **Когда появится запрос** — вставь содержимое `connect.html` и напиши `END`

8. **Повтори для** `setup.html`

9. **Готово!** Найди WiFi сеть `M5WeatherStation` и настраивай
